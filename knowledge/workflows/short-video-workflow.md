# Short Video Workflow

## Summary
For short-form output, start fast, test structure early, then refine only the strongest version.

## When to use
Use this workflow when you are making short videos for rapid iteration.

## Steps
1. Define one clear concept.
2. Generate a fast first pass.
3. Review pacing and clarity.
4. Refine only the best variation.
5. Export for the target platform.

## Tips
- Optimize for speed during the first round.
- Use stronger quality settings only after the concept works.

## Limitations
- This workflow favors speed over deep scene-building.
