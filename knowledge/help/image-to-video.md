# Image-to-Video

## Summary
Image-to-video helps you animate a visual starting point into motion output.

## When to use
Use this path when you already have a still image, frame, or visual concept and want to turn it into motion.

## Steps
1. Open the image-to-video workflow or feature area.
2. Upload or select your source image.
3. Add the motion direction or prompt guidance.
4. Choose the model and generation settings.
5. Run a short test generation.
6. Review movement, consistency, and composition.
7. Adjust and regenerate if needed.

## Tips
- Use a clean source image when possible.
- Keep motion guidance focused instead of overly broad.

## Limitations
- Strong motion changes can reduce visual consistency.
