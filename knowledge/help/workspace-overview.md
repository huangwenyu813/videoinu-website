# Workspace Overview

## Summary
The workspace is where you manage your project, creative inputs, generation steps, and exports.

## When to use
Use this guide when you want a quick mental model of how to move around the product.

## Steps
1. Open or create a project.
2. Review the current feature or workflow area.
3. Select the model or generation mode you want to use.
4. Update your inputs and generation settings.
5. Review outputs before exporting.

## Tips
- Keep one project focused on one deliverable when possible.
- Revisit the same workspace to refine results instead of restarting from scratch.

## Limitations
- Interface details may vary depending on the feature view you are in.
